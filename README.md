- 🔙 **Backend**: Java + Spring Boot 
- 🎨 **Frontend**: React + TypeScript

### 🧠 Current Features
✅ View product list
✅ Create new products via form
✅ Delete products
✅ Sort products alphabetically

- Form validation and error handling
- Search and pagination
